# Kanji CLI

CLI tool for fetching kanji info.

## Setup

Data files are not included in this repository. You need to download them manually:

- [KANJIDIC2](https://www.edrdg.org/kanjidic/kanjidic2.xml.gz)
- [KanjiVG](https://github.com/KanjiVG/kanjivg/releases/download/r20220427/kanjivg-20220427.xml.gz)

Place decompressed XML files in `data` directory in project root.

## Usage

Very simple:

```bash
kanji 件
```

Use `--help` to see available options.
