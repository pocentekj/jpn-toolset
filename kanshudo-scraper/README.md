# Kanshudo Scraper

> [Kanshudo](https://www.kanshudo.com/) is an excellent resource, and I use it constantly as part of my learning process. This tool simply helps me quickly add newly learned kanji to my personal catalog.

This project provides tools to extract data from Kanshudo for personal study.

The repository does NOT include any data from Kanshudo.

Users are responsible for ensuring their use complies with Kanshudo Terms of Service.

## Usage

Very simple:

```bash
kanshudo 件
```

Use `--help` to see available options.
